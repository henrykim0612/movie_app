## 2일차
1. this.state 변수의 값을 직접 컨트롤 하는것은 좋지 않다.  
   this.setState(current => ({count: current.count + 1})) 와 같이 current 함수를 이용해 state값을 수정한다.
   

2. setState를 호출하면 리액트는 새로운 state와 함께 클래스 컴포넌트의 render 함수를 호출한다.
